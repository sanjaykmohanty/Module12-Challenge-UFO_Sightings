# Module12-Challenge-UFO_Sightings

## Oerview
The purpose of this assignment is to provide an in-depth analysis of UFO sightings data. All the UFO shighting data is available to the user to look at. The user is allowed to use filter for multiple criteria such as date, city, state, country, and shape. HTML and JavaScript was used to set multiple filters and search for specific UFO sightings.

This report covers the steps followed to complete the technical analysis of filtering UFO sightings on multiple criteria.
